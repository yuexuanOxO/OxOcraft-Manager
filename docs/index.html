OxOcraft-Manager

Minecraft Server Management Tool
OxOcraft-Manager 是一款以 Flask 與 Python 開發的 Minecraft 伺服器管理工具。
提供伺服器啟動管理、世界備份、Google Drive 雲端備份、死亡紀錄、伺服器設定等功能。

主要功能

  • Minecraft Server 啟動與狀態管理
  • 本機世界備份
  • Google Drive 雲端備份
  • 自動備份系統
  • 死亡紀錄系統
  • 伺服器設定管理
  • Minecraft Query 狀態偵測

Google Drive OAuth 說明

OxOcraft-Manager 使用 Google OAuth 進行 Google Drive 備份功能授權。

本工具僅會要求：
  • Google Drive 檔案建立與存取權限
  • Google 帳號基本資訊
不會讀取或存取使用者完整 Google Drive 內容。
